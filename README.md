# Crypto Price Track
A real-time mobile application cryptocurrencies price tracker with charts using React Native, Rainbow.me, CoinGekoAPI, Axios

## Tech Stack

    - Front-end: React Native,Rainbow.me, Expo, Axios.
	- API: CoinGekoAPI
